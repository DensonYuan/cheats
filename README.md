# cheats
my navi cheats (with navi 2.x.x)

### 导入 cheat 文件
navi repo add https://github.com/DensonYuan/cheats

### Mac install navi:
`brew install navi`

cheat 文件默认位置: ~/Library/Application Support/navi/cheats

### Linux install navi:
- 下载 navi 二进制: https://github.com/denisidoro/navi/releases
- 解压二进制到 $PATH
